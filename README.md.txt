Movie Library

Автор: Иван Дорофеев

Описание
Программа «Movie Library» — это простое GUI-приложение для хранения информации о фильмах. В нем можно добавлять фильмы, фильтровать по жанру и году, а все данные сохраняются в JSON-файл.

Установка и запуск
1. Убедитесь, что у вас установлен Python 3.
2. Сохраните скрипт в файл, например, `movie_library.py`.
3. Запустите через командную строку

Использование
- Введите название, жанр, год выпуска и рейтинг фильма.
- Нажмите «Добавить фильм» для добавления.
- Используйте поля фильтра для отображения фильмов по жанру или году.
- Все данные сохраняются автоматически в файле `movies.json`.

Структура файла
- `movies.json`: хранит список добавленных фильмов.
